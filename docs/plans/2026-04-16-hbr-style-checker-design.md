# HBR 스타일 체커 설계 문서 (HBR Style Checker Design)

## 1. 개요
헤븐번즈레드(HBR)의 캐릭터 SS 및 레조넌스 스타일을 효율적으로 관리하고 필터링하는 웹 기반 체커 도구입니다. 기존 사이트의 가시성 문제를 해결하고, 최신 메타와 티어 정보를 직관적으로 보여주는 것을 목표로 합니다.

## 2. 주요 기능
- **SS/레조넌스 집중**: 의미 없는 S, A 랭크 스타일은 제외하고 실질적인 활용도가 높은 스타일만 관리합니다.
- **다중 필터링 시스템**:
  - **원소(Element)**: 화염, 빙결, 뇌전, 광, 암, 무속성.
  - **부대(Unit)**: 31A, 31B 등 부대별 모아보기.
  - **티어(Tier)**: game8 기준의 숫자 티어(0, 1, 2...) 적용.
  - **메타(Meta)**: 현재 유행하는 최강 조합 스타일 퀵 하이라이트.
- **시각적 효과**:
  - **기본**: 콤팩트한 아이콘 뷰.
  - **교복 스타일**: 풀돌 시 테두리를 다이아몬드 그라데이션으로 강조.
  - **한정/레조넌스**: 아이콘 상단에 뱃지 표시.
  - **필터링 연출**: 필터 제외 대상을 '흐리게(Dim)' 또는 '숨기기(Hide)' 토글 기능 제공.
- **사용자 데이터 저장**: 보유 현황 및 개인 설정은 브라우저의 `LocalStorage`에 저장.

## 3. 기술 스택
- **Frontend**: React (Vite)
- **Styling**: Vanilla CSS
- **Data**: 로컬 JSON (`styles.json`, `meta_teams.json`)
- **Deployment**: GitHub Pages (Static Hosting)

## 4. 데이터 구조 (예시)
```json
{
  "id": "ruka_light_res",
  "character_name": "카야모리 루카",
  "style_name": "광휘의 메시아",
  "isResonance": true,
  "isLimited": true,
  "isUniform": false,
  "unit": "31A",
  "element": "광",
  "tier": 0,
  "metaTags": ["광속성 파티", "범용 버퍼"]
}
```

## 5. UI/UX 원칙
- **가시성 우선**: 정보가 많아도 필터링을 통해 필요한 것만 돋보이게 처리.
- **반응형 디자인**: 모바일과 데스크톱 모두에서 편하게 체크 가능하도록 설계.
